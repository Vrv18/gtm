# Project Management

Central hub for tracking all GTM and operational initiatives.

## Project Stages

### Active Projects (`active/`)
Currently in progress with assigned owners and timelines.

### Backlog (`backlog/`) 
Approved projects waiting for resources or scheduling.

### Completed (`completed/`)
Finished projects with outcomes and learnings.

## Project Workflow

1. **Initiation** - Use `project-template.md` to define scope
2. **Planning** - Break down tasks and assign owners  
3. **Execution** - Regular updates and milestone tracking
4. **Completion** - Document results and move to completed folder
