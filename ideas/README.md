# Ideas & Brainstorms

This space is for capturing, developing, and tracking ideas across all GTM and operational areas.

## Structure

- `brainstorming/` - Active brainstorming sessions
- `backlog/` - Ideas for future consideration  
- `in-development/` - Ideas being actively explored
- `archived/` - Completed or discarded ideas

## How to Use

1. Start new ideas using the `idea-template.md`
2. Move ideas through stages: backlog → in-development → archived
3. Tag ideas with relevant departments and priority levels
4. Schedule regular idea review sessions
